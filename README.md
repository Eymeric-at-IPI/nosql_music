# NoSQL ( music )

Mini projet pour faire des requetes sur une base de donnée MongoDB.

## Prerequis

Une instance mongoDB en marche
Les données pour les collections **artistes** (`artistes.json`) et **albums** (`albums.json`) sont disponibles au format JSON à la racine du projet.

## HOW to use


```bash
git clone ...
```

```bash
pip install -r requirements.txt
```

```bash
python manage.py runserver
```

L'application est accessible a l'adress https://localhost:8000/hello/
